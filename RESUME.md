> **星见雅的双耳高塔** — 一款 Canvas 2D 物理堆叠小游戏，玩家控制移动角色将帽子精准堆叠在目标耳朵之上。独立完成前端架构与全量代码开发，实现双坐标系渲染系统、自适应摄像机平滑跟随、线段碰撞检测等核心机制。使用 TypeScript + Vite 构建，GitHub Actions 自动部署，支持桌面端与移动端。

## 详细版（适合项目经历展开描述）

**技术栈**：TypeScript、Canvas 2D、Vite、Web Audio API、GitHub Actions

**项目描述**：
独立设计并开发了一款基于 Canvas 2D 渲染的物理堆叠游戏。玩家控制屏幕上方按正弦曲线移动的角色，在恰当时机释放帽子，帽子受重力加速下落，落点与角色耳朵线段进行 X 轴范围匹配判定。游戏包含完整的菜单-游戏-暂停-结束状态机、渐进式难度曲线、剧情文案触发系统、素材编辑器以及最高分本地存储。

**核心实现**：

1. **双坐标系渲染架构** — 设计屏幕空间与世界空间分离的渲染管线，苍角固定在屏幕顶部不受摄像机影响，帽子在世界空间进行重力物理计算后映射回屏幕渲染，摄像机使用世界空间堆顶坐标平滑插值跟随，消除反馈循环

2. **线段碰撞检测算法** — 将 2D 碰撞简化为水平线段 X 范围判定，通过在角色图片上定义两条不可见耳朵线段实现高效落点匹配，支持运行时通过配置比例校准

3. **尖顶帽嵌套堆叠** — 通过可配置的重叠比例参数实现帽子层叠视觉效果，每层帽子仅暴露下半部分，模拟真实尖顶帽堆叠外观

4. **Web Audio 合成音效** — 使用 OscillatorNode 在代码中合成所有游戏音效（叠帽/掉落/失败/胜利），无需加载外部音频文件

5. **自动化 CI/CD** — 配置 GitHub Actions 工作流，推送代码后自动执行类型检查、生产构建与 GitHub Pages 部署

**项目亮点**：
- 零运行时依赖，构建产物 JS 仅 27KB（gzip 9KB）
- 自适应 Device Pixel Ratio，Retina 屏清晰渲染
- 完整的移动端触屏支持，全局点击/触摸事件处理
- localStorage 持久化最高分与素材调整参数
