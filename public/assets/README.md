# 游戏素材资源

将对应的 PNG 图片放入此目录，游戏启动时会自动加载。

## 需要的素材文件

| 文件名 | 说明 | 建议尺寸 |
| --- | --- | --- |
| `background.png` | 游戏背景图 | 1920×1080 |
| `miyabi.png` | 星见雅坐姿图（带透明通道） | 高度占游戏画面下 1/3 |
| `sofa.png` | 沙发图（带透明通道） | 宽度跨整个屏幕 |
| `leftEar.png` | 左耳底座图（带透明通道） | 宽 90px |
| `rightEar.png` | 右耳底座图（带透明通道） | 宽 90px |
| `hat.png` | 三角帽图（带透明通道） | 70×60 |
| `cangjiao.png` | 苍角图（带透明通道） | 64×80 |

## 加载规则

- Vite 会将 `public/` 下的文件原样输出到站点根目录，所以直接用 `assets/xxx.png` 引用即可。
- ResourceManager 启动时尝试 `assets/xxx.png` 加载这些资源；加载失败会回退到代码内 Canvas 绘制（仍可玩）。
- 你也可以在主菜单点击【自定义素材】按钮，临时上传其他图片覆盖（仅本次会话生效）。
